# Inventory Management System with CSV (Python)

## Overview
This project is a **console-based inventory management system** developed in Python.  
It allows users to manage products and persist data using CSV files.

The system implements:
- CRUD operations
- Inventory statistics
- CSV file handling (save & load)
- Input validation and error handling
- Modular programming
